Setup Guide

Get the bot running in under 5 minutes.

01

Get a Groq API Key

Sign up at console.groq.com — free tier, no card needed. Create an API key and copy it.

02

Clone & Install

git clone https://github.com/YOUR_USER/polymarket-bot.git
cd polymarket-bot
pip install -r requirements.txt
03

Configure Environment

Copy the example env file and paste your Groq key.

cp .env.example .env
GROQ_API_KEY your key from step 1
BANKROLL 20.0 (fake starting cash)
BET_AMOUNT 1.0 (per-bet size)
CONFIDENCE_THRESHOLD 0.65 (min AI confidence)
SLEEP_SECONDS 1800 (30 min cycles)
04

Run Locally

CLI-only mode (no dashboard):

python bot.py

With dashboard:

python server.py

Dashboard runs on localhost:8000.

05

Deploy to Koyeb

Push to GitHub, then deploy on koyeb.com:

  1. Create Service → GitHub → pick your repo
  2. Builder: Buildpack • Instance: Free nano
  3. Service type: Web
  4. Add GROQ_API_KEY in environment variables
  5. Deploy — bot + dashboard run 24/7
